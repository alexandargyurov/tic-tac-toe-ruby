# Tic Tac Toe - Ruby Rack

This is a simple Rack server running a simple Tic Tac Toe game made with Ruby. 

Play it here:
https://tic-tac-toe-ruby.herokuapp.com/


## How to run:

```sh
$ cd tic-tac-toe-ruby
$ bundle install
$ rackup

http://localhost:9292
```
